# MyAniVault | An anime catalog of some favorite animes I've watched!

[Website](https://lorenz0l.github.io/Snapchat-EA-Catalog-Website-MyAniVault/)

A catalog website built for the Snap Engineering Academy Stage 2 project assessment.

![Preview](assets/preview.png)

## About
AniVault is a catalog of must-watch anime series. Users can search, filter by genre, and sort by rating or year to explore the collection.

## Features
- Search by title or studio
- Filter by genre
- Sort by rating, year, or title

## Data
18 anime series with data including title, genre, year, rating, episodes, studio, status, and description. Data sourced from [MyAnimeList](https://myanimelist.net).

## Built With
HTML, CSS, JavaScript
